## Getting Started

To get you started you can simply clone the Survey repository and install the dependencies:


```
npm install
```

```
bower install
```

To start the server run
```
grunt
```
Now browse to the app at `http://localhost:8000/`.

```
npm test
```

To run the jasmine specs.
